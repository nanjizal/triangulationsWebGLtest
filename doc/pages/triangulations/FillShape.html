triangulations API documentation

class FillShape

package triangulations

Basic structure used to store a Shape. vertices are coordinates of Vector2 Edges are two vertices indices p and q to define an edge * Faces is array of array of Faces ( polygons ).

Constructor

new ()

constructor empty but can be overridden by abstracts that might create a new shape.

Variables

edges:Edges

abstract of Array, each edge holds two indicies of vertices p and q.

faces:Array<Array<Face>>

array of array of faces, so for instatce each array of face would be a letter

 *  in the Array<Face> the first Face is assumed the polygon outline and the other faces holes within

vertices:Vertices

abstract of Array used to hold shape coordinates external at first and then later prehaps internal

Methods

inline clone ():FillShape

clones a FillShape, cloning it's properties and creating a new FillShape with them.

inline cloneFaces ():Array<Array<Face>>

clone helper for cloning faces property

inline fit (w:Float, h:Float, margin:Float = 10):Void

fit the coordinates within width w, and height h with a outer margin all round

Parameters:

w

bounding width to fit coordinates within

h

bounding height to fit coordinates within

margin

margin within bounding box

inline scale (s:Float):Void

scales the vertices coordinates

Parameters:

scale

scale factor

inline set_fixedExternal (val:Bool):Void

set edges to all be edges to be external fixed if true and not if false

Parameters:

val

true to set edges to be fixed and external, normally set after creating a new Fill Shape

inline translate (dx:Float, dy:Float):Void

translates the coordinates by dx and dy amount.

Parameters:

dx

translate on x axis by dx amount

dy

translate on y axis by dy amount
