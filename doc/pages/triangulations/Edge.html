class Edge

Edge holds two vertex indices p and q Also can be fixed and/or external

new (p_:Null<Int>, q_:Null<Int>)

create edge from to vertices indices

Parameters:

p

first index

q

second index

Variables

edge that can't be changed ( flipped for instance ).

q:Null<Int> = null

second vertex index
